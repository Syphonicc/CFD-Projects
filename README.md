# OpenFOAM CFD Projects

Computational fluid dynamics simulations built while 
learning OpenFOAM on Fedora Linux.

## Cases
- Pipe Elbow -> incompressible viscous flow through a 90° bend
- Lid Driven Cavity -> incompressible flow in a two-dimensional square domain
- motorBike -> external aerodynamics around a motorbike and rider using snappyHexMesh + simpleFoam (steady-state RANS, k-omega SST, ~353k cells)
